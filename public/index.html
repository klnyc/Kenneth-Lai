Projects
iDestination
iDestination is a travel planning desktop/mobile optimized single-page application which allows users to keep track of places that they have traveled to in the past and places that they plan to visit in the future to plan trips more efficiently. Users can create an account to search, add, delete and filter locations by category on an integrated map, save their home location and view real-time weather of any city. Designed a user database schema using Cloud Firestore to store user information and saved locations.
JavaScript, Node.js, React, Redux, Firebase, Webpack, Google Maps API, Open Weather API, HTML, CSS
Jukebox Center
Jukebox Center is an e-commerce single-page application website that sells music albums. Users can add albums to their cart after creating an account or as a guest, update their account information, view their order history and filter albums by genre. Album stock quantities will update in the database after each purchase. Designed a user/album/cart database schema in PostgreSQL, connected through REST API using Express.
JavaScript, Node.js, React, Redux, Express, PostgresSQL, Sequelize, Webpack, Heroku, HTML, CSS
